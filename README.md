# Vocabulary-checklist (WIP)
Ready for a small test?

Welcome to the Vocabulary checker (by @Dalufi)!
This is an open-source (MIT applies) Vocabulary tool you can practice with in practically any language. 

Tired of failing vocabulary checks in school or do you just want to challenge yourself? Well, with this simple tool you can practice as much as you want! You can create your OWN vocabulary list in ANY language but that's not all, you can even export and import them for your friends. Now you have your list but what do you do with it? Study! With the vocabulary checker you get a word from one language and an input field for the same word but in the other language. Don't worry, if you type in the wrong term you can always try again in the "refresh round". After completing a vocabulary check you can try again but this time, with the words you didn't know. Have fun!
